发表于 2017-11-03 | 分类于 Android Studio介绍

通过Android studio在Github上提交及更新代码

一 假设你有Mac和PC两台电脑

二 先在Mac的AS中创建项目并上传Github

1 创建项目
2 修改主.gitignore
*.iml
.gradle
local.properties
.idea
.DS_Store
build
3 确定git已安装,点击test按钮如果出现 Git executed successfully 对话框说明配置成功

4 确定github已连接

填入如下信息： 

Host:github.com
Login:你的github账户名
Password：你的github账户密码
填完之后点击test按钮，如果出现如下对话框说明配置成功
